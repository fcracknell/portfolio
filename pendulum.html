Swing-Up and LQR Control Simulation of an Inverted Pendulum

This project simulates an inverted pendulum on a cart using a two-phase control approach: an energy-based swing-up algorithm followed by Linear Quadratic Regulation (LQR) for stabilisation. The simulation environment is implemented in Python using Pymunk and Pygame for physics and visualization, while the control design is developed in MATLAB.

View Source Code on GitHub

Methodology

The pendulum is first brought near the upright equilibrium through energy shaping, where the control input is based on the difference between the current energy and the target energy of the upright state. Once close to vertical, the system transitions to a full-state feedback LQR controller designed using MATLAB's state-space representation.

Results

The LQR controller successfully stabilizes the pendulum after the energy-based swing-up phase. Control effort and state contributions are visualized using MATLAB plots, highlighting the role of each state in the feedback law.
